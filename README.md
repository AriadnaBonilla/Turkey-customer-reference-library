# SAP Turkey — Customer Reference Library

> ⚠️ **INTERNAL — SAP and External Parties under NDA Only**

An interactive HTML reference library of **155+ Turkish SAP Cloud ERP customers** across 20 industries.

## What's inside

| File | Description |
|------|-------------|
| `index.html` | Self-contained interactive library — open directly in any browser |

## Features

- **Search** by company name
- **Filter** by industry (Retail, Automotive, Manufacturing, Energy, Fashion, Aviation, Port Operations, and more)
- **159 company logos** extracted from SAP Turkey presales slides and embedded as base64 — works fully offline
- **Download** any logo as a PNG with one click
- Stats bar showing total customers and active filters

## How to use

1. Clone or download this repository
2. Open `index.html` in any modern browser — no server or internet connection required
3. To publish via GitHub Pages: go to **Settings → Pages → Source: main / root**

## Data

All customers are Turkish SAP Cloud ERP references. Content is INTERNAL and subject to NDA.
